6. Calendar Connection

Location: Settings → Calendars → Connections

  1. Select user from "My Staff" dropdown
  2. Click "+ Add New" under Connected Calendars
  3. Select Google
  4. Sign in with hello@slatesystems.io
  5. Configure:
    • Linked Calendar: hello@slatesystems.io
    • Conflict Calendars: hello@slatesystems.io
Both Dan A and Rick F should be connected to hello@slatesystems.io calendar. This is the shared Slate calendar that syncs to Dan and Ricardo's personal calendars.

7. Workflow Assignments

Update these workflow nodes when user structure changes:

WorkflowNodeAssign To
1. Lead Form → New OpportunityAssign to UserDan A
1. Client Paid → Send Onboarding InfoAssign to UserRick F

8. Checklist: New User Setup

  1. Create user with basic info (name, email, phone, password)
  2. Set profile picture (Slate padded icon)
  3. Set User Role to "User" (not Admin)
  4. Configure permissions per tables above
  5. Configure notification settings
  6. Configure call & voicemail settings
  7. Connect calendar (hello@slatesystems.io)
  8. Update workflow "Assign to User" node if needed
  9. Test login as user
